# oomc_fun

これは [Add-Ons for Bedrock Edition](https://www.minecraft.net/addons/) について学ぶための、実験的なプロジェクトです。
いくつかのブロックやレシピなどの、ゲーム要素を追加します。

## Supported Locales

- en_US
- ja_JP

## Setup Addon

[アドオンのセットアップ | Setup Addon](docs/setup_addon.md)

## Build

- [Node.js](https://nodejs.org/) >= 14

```
npm install
npm run build
```
